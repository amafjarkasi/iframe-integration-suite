iframe-integration-suite Parent Demo

This demo shows different iframe integration scenarios:

  • Same-origin: Direct DOM access and extraction
  • Cross-origin cooperative: postMessage RPC communication
  • Cross-origin non-cooperative: Proxy-based access

Scenario 1: Same-Origin Extraction

Load a same-origin page and extract its content directly.

Scenario 2: Cross-Origin Cooperative

Communicate with a cooperative child page using postMessage RPC.

Scenario 3: Proxy-Based Access

Access non-cooperative third-party pages through the proxy server.
